# Scenario

What is one "Epoch" in training?

## Options

- `One second of training.`
- `One pass through the entire dataset.`
- `One update of the weights.`
- `One prediction.`

## Correct Answer

```
One pass through the entire dataset.
```

## Explanation

If you have 1,000 images, one epoch means the model has seen all 1,000 images once.
