# Scenario

What is the role of the Input Layer?

## Options

- `To make the final prediction.`
- `To receive the raw data (e.g., image pixels).`
- `To perform complex calculations.`
- `To calculate the error.`

## Correct Answer

```
To receive the raw data (e.g., image pixels).
```

## Explanation

The input layer is the entry point. It has one neuron for every feature in the dataset (e.g., 784 neurons for a 28x28 image).
