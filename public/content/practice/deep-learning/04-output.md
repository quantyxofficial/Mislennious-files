# Scenario

If you are building a binary classifier (Cat or Dog), how many neurons should be in the Output Layer?

## Options

- `1 Neuron (0 or 1)`
- `100 Neurons`
- `28 Neurons`
- `0 Neurons`

## Correct Answer

```
1 Neuron (0 or 1)
```

## Explanation

For binary classification, a single neuron with a Sigmoid activation (0 to 1) is sufficient.
