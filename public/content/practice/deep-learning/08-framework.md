# Scenario

What is the relationship between Keras and TensorFlow?

## Options

- `They are competitors.`
- `Keras is a high-level API wrapper running on top of TensorFlow.`
- `TensorFlow is a wrapper for Keras.`
- `Keras is for Java, TensorFlow is for Python.`

## Correct Answer

```
Keras is a high-level API wrapper running on top of TensorFlow.
```

## Explanation

TensorFlow handles the math engine (graphs, gradients). Keras provides the user-friendly code (`model.fit`).
