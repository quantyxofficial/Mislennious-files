# Scenario

What is the purpose of the Loss Function (e.g., MSE, Cross-Entropy)?

## Options

- `To make the model faster.`
- `To quantify how wrong the model's predictions are compared to the truth.`
- `To prevent overfitting.`
- `To visualize the data.`

## Correct Answer

```
To quantify how wrong the model's predictions are compared to the truth.
```

## Explanation

The network aims to minimize this Loss value closer to 0 over time.
