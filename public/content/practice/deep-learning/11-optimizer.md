# Scenario

What does an Optimizer (like Adam or SGD) do?

## Options

- `It defines the architecture.`
- `It actually performs the weight updates to minimize loss.`
- `It speeds up the GPU.`
- `It selects the best features.`

## Correct Answer

```
It actually performs the weight updates to minimize loss.
```

## Explanation

The Loss function calculates *how* wrong we are. The Optimizer decides *how* to move (step size) to fix it.
