# Scenario

In a CNN, what is a "Filter" (or Kernel)?

## Options

- `A small matrix that slides over the image to detect features (edges, shapes).`
- `A tool to blur the image.`
- `A way to remove color.`
- `The output prediction.`

## Correct Answer

```
A small matrix that slides over the image to detect features (edges, shapes).
```

## Explanation

Filters are the "eyes" of the CNN. They learn to activate when they see specific patterns like vertical lines or circles.
