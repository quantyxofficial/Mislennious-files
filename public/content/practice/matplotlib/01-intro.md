# Scenario

What is Matplotlib primarily used for?

## Options

- `Machine Learning Training`
- `Database Management`
- `2D Plotting and Visualization`
- `Web Scraping`

## Correct Answer

```
2D Plotting and Visualization
```

## Explanation

Matplotlib is the "grandfather" of Python visualization libraries, designed originally to emulate MATLAB's plotting capabilities.
