# Scenario

Which function creates a horizontal bar chart?

## Options

- `ax.bar(orientation='h')`
- `ax.bar_horizontal()`
- `ax.barh()`
- `ax.hbar()`

## Correct Answer

```
ax.barh()
```

## Explanation

`barh` is the specific function for horizontal bars. It's often preferred for categorical data with long labels (like country names).
