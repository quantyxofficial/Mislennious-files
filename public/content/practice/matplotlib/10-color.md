# Scenario

Which of the following is NOT a valid color specification in Matplotlib?

## Options

- `'red'`
- `'#FF0000'`
- `(1.0, 0.0, 0.0)`
- `'col:red'`

## Correct Answer

```
'col:red'
```

## Explanation

Matplotlib accepts names (`'red'`), Hex (`'#FF0000'`), RGB Tuples (`(1,0,0)`), and Tableau colors (`'tab:red'`). `'col:red'` is invalid.
