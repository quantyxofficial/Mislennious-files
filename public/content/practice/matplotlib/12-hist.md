# Scenario

In `ax.hist(data, bins=50)`, what does `bins=50` control?

## Options

- `The height of the bars.`
- `The number of buckets (intervals) the data is divided into.`
- `The number of data points to plot.`
- `The width of the plot.`

## Correct Answer

```
The number of buckets (intervals) the data is divided into.
```

## Explanation

Bins define resolution. 50 bins means the range is divided into 50 equal intervals.
