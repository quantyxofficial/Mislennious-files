# Scenario

After defining your plot with `plt.plot()`, command must you run to actually make the window appear?

## Options

- `plt.render()`
- `plt.display()`
- `plt.show()`
- `plt.draw()`

## Correct Answer

```
plt.show()
```

## Explanation

`plt.show()` triggers the event loop that renders the window.
