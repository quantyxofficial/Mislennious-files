# Scenario

What import is required to enable 3D plotting in Matplotlib?

## Options

- `import matplotlib.3d`
- `from mpl_toolkits.mplot3d import Axes3D`
- `import matplotlib.pyplot as plt3d`
- `No import needed, just set projection='3d'`

## Correct Answer

```
from mpl_toolkits.mplot3d import Axes3D
```

## Explanation

While newer versions are more lenient, explicitly importing `Axes3D` from `mpl_toolkits` registers the 3D projection correctly with Matplotlib.
