# Scenario

Which command draws a simple line chart connecting (x, y) points?

## Options

- `ax.line(x, y)`
- `ax.plot(x, y)`
- `ax.chart(x, y)`
- `ax.draw(x, y)`

## Correct Answer

```
ax.plot(x, y)
```

## Explanation

`plot()` is the default function for line charts.
