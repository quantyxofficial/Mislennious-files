# Scenario

In the Object-Oriented interface, what does the `Figure` object represent?

## Options

- `The lines on the chart.`
- `The entire window/page holding all plots.`
- `The x and y axis labels.`
- `The data points.`

## Correct Answer

```
The entire window/page holding all plots.
```

## Explanation

The `Figure` is the top-level container (The Canvas). The `Axes` are the actual plots drawn onto that canvas.
