# Scenario

How do you turn on the grid lines?

## Options

- `ax.grid(True)`
- `ax.show_grid()`
- `plt.grid_on()`
- `ax.set_grid('on')`

## Correct Answer

```
ax.grid(True)
```

## Explanation

`ax.grid(True)` enables the grid. You can also pass `linestyle`, `alpha`, and `color` to customize it.
