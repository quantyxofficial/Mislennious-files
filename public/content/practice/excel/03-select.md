# Scenario

Which keyboard shortcut selects **All** data in the current region?

## Options

- `Ctrl + C`
- `Ctrl + A`
- `Ctrl + S`
- `Alt + F4`

## Correct Answer

```
Ctrl + A
```

## Explanation

`Ctrl + A` selects the contiguous range of data around your active cell.
