# Scenario

What is the primary purpose of Power Query (Get & Transform)?

## Options

- `To make 3D charts.`
- `To write VBA macros.`
- `To extract, transform, and clean data before loading it into Excel.`
- `To password protect the file.`

## Correct Answer

```
To extract, transform, and clean data before loading it into Excel.
```

## Explanation

Power Query is Excel's built-in ETL (Extract, Transform, Load) tool. It replaces complex formulas and macros for data cleaning.
