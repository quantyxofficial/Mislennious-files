# Scenario

`=IF(A1 > 50, "Pass", "Fail")`
If A1 is 50, what acts is returned?

## Options

- `"Pass"`
- `"Fail"`
- `Error`
- `50`

## Correct Answer

```
"Fail"
```

## Explanation

The condition is `>` (Strictly Greater Than). 50 is not greater than 50. If it were `>= 50`, it would pass.
