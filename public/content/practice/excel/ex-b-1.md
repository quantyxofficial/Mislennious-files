# Scenario

You have a column of sales figures in cells A1 through A10. You need to calculate the total sales.

## Options

- `=ADD(A1:A10)`
- `=SUM(A1:A10)`
- `=TOTAL(A1:A10)`
- `=COUNT(A1:A10)`

## Correct Answer

```
=SUM(A1:A10)
```

## Explanation

The `SUM` function adds all numbers in a range of cells.
