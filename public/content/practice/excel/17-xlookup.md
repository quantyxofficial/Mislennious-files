# Scenario

What is a major advantage of XLOOKUP over VLOOKUP?

## Options

- `XLOOKUP can look to the left (return a value from a column before the lookup column).`
- `XLOOKUP defaults to Exact Match.`
- `XLOOKUP doesn't break if you insert columns.`
- `All of the above.`

## Correct Answer

```
All of the above.
```

## Explanation

XLOOKUP was designed to fix every annoyance of VLOOKUP. It is bidirectional, safer, and cleaner.
