# Scenario

You have a long table and when you scroll down, you lose the header row. What feature fixes this?

## Options

- `Lock Cells`
- `Protect Sheet`
- `Freeze Panes`
- `Split Window`

## Correct Answer

```
Freeze Panes
```

## Explanation

Freeze Panes (View Tab) locks the top row or first column so they remain visible while scrolling.
