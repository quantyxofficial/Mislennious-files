# Scenario

You want to count the number of cells that contain text names (e.g., "John", "Alice"). Which function should you use?

## Options

- `COUNT`
- `COUNTA`
- `COUNTIF`
- `SUM`

## Correct Answer

```
COUNTA
```

## Explanation

`COUNT` only counts numbers. `COUNTA` counts "Anything" (not empty). `COUNTIF` requires a specific condition.
