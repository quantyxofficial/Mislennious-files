# Scenario

Which menu location allows you to deduplicate a list?

## Options

- `Data -> Remove Duplicates`
- `Home -> Conditional Formatting -> Duplicates`
- `Formulas -> Dedupe`
- `View -> Unique`

## Correct Answer

```
Data -> Remove Duplicates
```

## Explanation

The "Remove Duplicates" button is a core tool in the Data tab. Conditional Formatting only *highlights* them.
