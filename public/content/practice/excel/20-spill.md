# Scenario

You type `=UNIQUE(A1:A100)` in cell B1. The results automatically fill cells B1 to B10. What is this behavior called?

## Options

- `AutoFill`
- `Spilling`
- `Overflow`
- `Ghosting`

## Correct Answer

```
Spilling
```

## Explanation

In Modern Excel with Dynamic Arrays, formulas that return multiple values "spill" into adjacent empty cells.
