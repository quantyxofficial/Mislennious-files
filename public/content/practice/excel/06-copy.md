# Scenario

What is the universal shortcut for Copy and Paste?

## Options

- `Ctrl + X / Ctrl + V`
- `Ctrl + C / Ctrl + V`
- `Ctrl + C / Ctrl + P`
- `Alt + C / Alt + V`

## Correct Answer

```
Ctrl + C / Ctrl + V
```

## Explanation

Standard across all operating systems. `Ctrl + X` is Cut. `Ctrl + P` is Print.
