# Scenario

What is the difference between an Excel Workbook and a Worksheet?

## Options

- `They are the same thing.`
- `A Workbook contains multiple Worksheets (tabs).`
- `A Worksheet contains multiple Workbooks.`
- `A Workbook is for charts only.`

## Correct Answer

```
A Workbook contains multiple Worksheets (tabs).
```

## Explanation

The Workbook is the file (`.xlsx`). The Worksheets are the individual pages/tabs at the bottom.
