# Scenario

You've done 3 hours of work. How do you save immediately?

## Options

- `Ctrl + P`
- `Ctrl + S`
- `Alt + F4`
- `F1`

## Correct Answer

```
Ctrl + S
```

## Explanation

Save early, save often.
