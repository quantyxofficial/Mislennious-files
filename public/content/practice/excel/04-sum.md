# Scenario

Which formula adds up the values in cells A1, A2, and A3?

## Options

- `=ADD(A1:A3)`
- `=SUM(A1:A3)`
- `=TOTAL(A1:A3)`
- `=PLUS(A1:A3)`

## Correct Answer

```
=SUM(A1:A3)
```

## Explanation

`SUM` is the universal function for addition. There is no `ADD` or `TOTAL` function.
