# Scenario

After training, how do you generate predictions for new data `X_new`?

## Options

- `model.guess(X_new)`
- `model.predict(X_new)`
- `model.calc(X_new)`
- `model.fit(X_new)`

## Correct Answer

```
model.predict(X_new)
```

## Explanation

`predict` uses the learned rules to output answers for new inputs.
