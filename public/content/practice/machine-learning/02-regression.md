# Scenario

What implies a "Regression" problem?

## Options

- `Predicting a continuous number (e.g., Price, Temperature).`
- `Predicting a Category (e.g., Cat/Dog).`
- `Grouping similar items.`
- `Finding anomalies.`

## Correct Answer

```
Predicting a continuous number (e.g., Price, Temperature).
```

## Explanation

If the output is a quantity (how much, how many), it is Regression.
