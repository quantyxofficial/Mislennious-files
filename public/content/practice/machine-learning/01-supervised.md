# Scenario

What distinguishes Supervised Learning from Unsupervised Learning?

## Options

- `Supervised Learning requires labeled data (Input + Correct Output).`
- `Supervised Learning is faster.`
- `Supervised Learning doesn't need data.`
- `Unsupervised Learning uses labels.`

## Correct Answer

```
Supervised Learning requires labeled data (Input + Correct Output).
```

## Explanation

In Supervised learning, you act as a "teacher", providing the model with the answers (labels) so it can learn the rules.
