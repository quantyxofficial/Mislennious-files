# Scenario

How does a Random Forest make a final prediction for classification?

## Options

- `It picks the result of the first tree.`
- `It takes the majority vote of all the trees.`
- `It averages the depth of the trees.`
- `It picks the most complex tree.`

## Correct Answer

```
It takes the majority vote of all the trees.
```

## Explanation

It is an "Ensemble" method. The wisdom of the crowd (100 trees) beats the individual expert (1 tree).
