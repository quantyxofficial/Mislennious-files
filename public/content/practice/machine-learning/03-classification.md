# Scenario

What implies a "Classification" problem?

## Options

- `Predicting which category an item belongs to (Tagging).`
- `Predicting a future stock price.`
- `Sorting a list.`
- `Calculating the average.`

## Correct Answer

```
Predicting which category an item belongs to (Tagging).
```

## Explanation

If the output is a label or class (Yes/No, Spam/Ham, Red/Blue), it is Classification.
