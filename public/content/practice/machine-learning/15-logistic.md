# Scenario

What is the output range of the Logistic Sigmoid function?

## Options

- `-Infinite to +Infinite`
- `0 to 1`
- `-1 to 1`
- `0 to 100`

## Correct Answer

```
0 to 1
```

## Explanation

The sigmoid function "squashes" any input number into the range [0, 1], representing a probability.
