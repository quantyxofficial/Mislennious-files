# Scenario

Which of the following best describes Supervised Learning?

## Options

- `Training a model on labeled data`
- `Finding hidden structures in unlabeled data`
- `Learning through trial and error`
- `Hardcoding rules for decision making`

## Correct Answer

```
Training a model on labeled data
```

## Explanation

Supervised learning involves training a model on a dataset where the target output (label) is known. The model learns the mapping from inputs to outputs.
