# Scenario

What does an R-Squared (R2) score of 1.0 mean?

## Options

- `The model is 100% accurate.`
- `The model explains 100% of the variance in the target variable.`
- `The model has 1.0 error.`
- `The model is overfitting.`

## Correct Answer

```
The model explains 100% of the variance in the target variable.
```

## Explanation

R2 measures goodness of fit. 1.0 means the model perfectly fits the data (which usually implies Overfitting in the real world, but mathematically it means perfect variance explanation).
