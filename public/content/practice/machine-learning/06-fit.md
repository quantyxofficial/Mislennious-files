# Scenario

In Scikit-Learn, what does `model.fit(X, y)` do?

## Options

- `It predicts the output.`
- `It trains the model (learns the patterns from X and y).`
- `It cleans the data.`
- `It saves the file.`

## Correct Answer

```
It trains the model (learns the patterns from X and y).
```

## Explanation

`fit` is the universal command to start the learning process.
