# Scenario

In standard ML notation (X, y), what is `y`?

## Options

- `The Comparison variable.`
- `The Input Features.`
- `The Target / Label (Assumption).`
- `The Row Index.`

## Correct Answer

```
The Target / Label (Assumption).
```

## Explanation

`X` is the matrix of features (Questions). `y` is the vector of targets (Answers).
