# Scenario

What is "Mean Imputation"?

## Options

- `Deleting any row with a missing value.`
- `Replacing missing values with the average of that column.`
- `Replacing missing values with 0.`
- `Replacing missing values with the most frequent value.`

## Correct Answer

```
Replacing missing values with the average of that column.
```

## Explanation

It's a simple strategy to preserve data volume. However, it can distort the distribution if data is not missing at random.
