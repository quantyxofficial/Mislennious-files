# Scenario

Which of the following is a common ratio for Splitting Training and Testing data?

## Options

- `50% Train, 50% Test`
- `80% Train, 20% Test`
- `20% Train, 80% Test`
- `100% Train, 0% Test`

## Correct Answer

```
80% Train, 20% Test
```

## Explanation

The 80/20 (Pareto) rule is standard. 70/30 or 90/10 are also common. You need enough data to train, but enough data to validly test.
