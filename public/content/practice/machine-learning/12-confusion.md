# Scenario

In a Confusion Matrix, what is a "False Positive" (Type I Error)?

## Options

- `The model predicted YES, and the truth was YES.`
- `The model predicted YES, but the truth was NO.`
- `The model predicted NO, but the truth was YES.`
- `The model crashed.`

## Correct Answer

```
The model predicted YES, but the truth was NO.
```

## Explanation

Example: Testing a healthy patient (Negative) and telling them they have cancer (Positive). It's a false alarm.
