# Scenario

How do you create a simple 1D NumPy array from a Python list `[1, 2, 3]`?

## Options

- `np.create([1, 2, 3])`
- `np.array(1, 2, 3)`
- `np.array([1, 2, 3])`
- `np.list_to_array([1, 2, 3])`

## Correct Answer

```
np.array([1, 2, 3])
```

## Explanation

`np.array()` is the fundamental constructor. It takes a sequence-like object (like a list) and converts it into an ndarray.
