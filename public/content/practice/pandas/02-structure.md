# Scenario

What is the relationship between a Series and a DataFrame?

## Options

- `A Series is a 2D table, a DataFrame is 3D.`
- `A DataFrame is a collection of Series objects sharing the same index.`
- `A Series is for text, a DataFrame is for numbers.`
- `They are unrelated.`

## Correct Answer

```
A DataFrame is a collection of Series objects sharing the same index.
```

## Explanation

Think of a DataFrame like a Table. Each column in that table is a **Series**.
