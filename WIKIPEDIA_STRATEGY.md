# How to Get Wikipedia Coverage for KaizenStat

## 📖 **IMPORTANT: Wikipedia Policies First**

Before we start, understand **Wikipedia's core requirements**:

1. **Notability** - Must be notable enough for Wikipedia
2. **Reliable Sources** - Need external sources (news, blogs, journals)
3. **No Original Research** - Can't write from your own bias
4. **No Promotion** - Can't use Wikipedia as marketing
5. **Verifiability** - Must be verifiable from independent sources

## 🚫 **WHAT NOT TO DO**

### **Common Wikipedia Mistakes**

❌ **Don't:**
- Create article yourself if not notable yet
- Use promotional language ("best", "amazing")
- Link to your own website as source
- Include unreliable sources
- Write like a press release
- Exaggerate numbers or achievements
- Ignore Wikipedia guidelines
- Use Wikipedia for marketing
- Add information without sources
- Edit related articles to include yourself

✅ **Do:**
- Build notability first with media coverage
- Use neutral, factual language
- Cite only reliable independent sources
- Follow Wikipedia style guides
- Let others create the article
- Support Wikipedia with accurate info
- Help other articles improve
- Be honest about your project

## 🎓 **WIKIPEDIA GUIDELINES TO KNOW**

### **Notability Standard**
Wikipedia says a project is notable if:
> "The subject has been the subject of significant coverage in reliable sources that are independent of the subject."

### **Three Key Policies**
1. **Verifiability** - Must be verifiable from reliable sources
2. **Neutral Point of View** - No promotional language
3. **No Original Research** - Can't use your own analysis

### **Reliable Sources Definition**
Wikipedia accepts:
- Published books
- Scholarly journals
- News organizations
- Magazines
- Academic institutions
- Industry reports

### **What Counts as Promotion**
Don't use:
- "Best", "Amazing", "Revolutionary"
- "Industry leader", "Most trusted"
- "Everyone should use"
- Links to your own site as source
