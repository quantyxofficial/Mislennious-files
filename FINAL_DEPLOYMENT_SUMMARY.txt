HOW TO DEPLOY TO VERCEL
════════════════════════════════════════════════════════════════════════════════

STEP 1: Go to https://vercel.com/import

STEP 2: Connect GitHub
   • Select "GitHub" as deployment source
   • Authenticate your account
   • Choose repository: quantyxofficial/Mislennious-files
   • Vercel auto-detects Vite framework

STEP 3: Configure Environment Variables
   In Vercel Project Settings → Environment Variables, add:
   
   VITE_SUPABASE_URL = https://pdqytkydmbvepwiddwzb.supabase.co
   VITE_SUPABASE_PUBLISHABLE_KEY = [from Supabase]
   VITE_CERT_SUPABASE_URL = https://lmjvkfyabdcainqkmmwt.supabase.co
   VITE_CERT_SUPABASE_KEY = [from Supabase]
   GEMINI_API_KEY = [from Google AI Studio]
   
   Set all variables for: Production, Preview, Development

STEP 4: Deploy
   • Click "Deploy"
   • Wait 5-10 minutes for build to complete
   • Get live URL automatically

STEP 5: Verify
   • Check homepage loads correctly
   • Verify all routes work
   • Check SEO tags in HTML
   • Test Supabase integration
